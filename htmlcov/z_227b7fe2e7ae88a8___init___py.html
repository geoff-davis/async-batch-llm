1"""Batch LLM processing utilities for handling bulk LLM requests. 

2 

3This module provides a flexible framework for processing multiple LLM requests 

4efficiently using a strategy pattern for provider-agnostic LLM integration. 

5 

6Key features: 

7- Strategy pattern for any LLM provider (OpenAI, Anthropic, Google, LangChain, custom) 

8- Built-in strategies: PydanticAIStrategy, GeminiStrategy, OpenAIStrategy, OpenRouterStrategy 

9- Built-in models: GeminiModel, GeminiCachedModel, OpenAIModel, OpenRouterModel 

10- Provider-agnostic error classification 

11- Pluggable rate limit strategies 

12- Middleware pipeline for extensibility 

13- Observer pattern for monitoring 

14- Configuration-based setup 

15 

16Example: 

17 >>> from async_batch_llm import ( 

18 ... ParallelBatchProcessor, 

19 ... ProcessorConfig, 

20 ... LLMWorkItem, 

21 ... PydanticAIStrategy, 

22 ... ) 

23 >>> from pydantic_ai import Agent 

24 >>> 

25 >>> agent = Agent("openai:gpt-4o-mini", result_type=MyOutput) 

26 >>> strategy = PydanticAIStrategy(agent=agent) 

27 >>> config = ProcessorConfig(max_workers=5, timeout_per_item=60.0) 

28 >>> 

29 >>> async with ParallelBatchProcessor(config=config) as processor: 

30 ... await processor.add_work(LLMWorkItem( 

31 ... item_id="item_1", 

32 ... strategy=strategy, 

33 ... prompt="Process this", 

34 ... )) 

35 ... result = await processor.process_all() 

36 

37Type Aliases: 

38 For convenience, type aliases are provided to reduce verbosity: 

39 

40 - ``SimpleBatchProcessor[T]``: Processor with string input, output type T, no context 

41 Equivalent to ``ParallelBatchProcessor[str, T, None]`` 

42 

43 - ``SimpleWorkItem[T]``: Work item with string input, output type T, no context 

44 Equivalent to ``LLMWorkItem[str, T, None]`` 

45 

46 - ``SimpleResult[T]``: Result with output type T, no context 

47 Equivalent to ``WorkItemResult[T, None]`` 

48 

49 Example using type aliases: 

50 >>> from async_batch_llm import SimpleBatchProcessor, SimpleWorkItem 

51 >>> 

52 >>> async with SimpleBatchProcessor[MyOutput](config=config) as processor: 

53 ... await processor.add_work(SimpleWorkItem[MyOutput]( 

54 ... item_id="item_1", 

55 ... strategy=strategy, 

56 ... prompt="Process this", 

57 ... )) 

58"""
